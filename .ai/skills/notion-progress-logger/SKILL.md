# 노션 진행 기록

## 기록 대상

아래 내용은 노션 하위 페이지로 기록한다.

- AI 역인터뷰
- 최종 또는 수정된 계획
- 일자별 진행 로그
- 에러와 해결 방법
- GitHub 커밋/푸시 기록
- 배포 기록
- 포트폴리오 설명 초안

## 작성 방식

- 한국어로 작성한다.
- 나중에 이어서 작업하기 쉽도록 사실 중심으로 기록한다.
- 필요하면 경로, 명령어, 다음 단계를 함께 남긴다.
