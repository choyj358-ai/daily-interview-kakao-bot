# 카카오 메시지 연동

## 1차 버전 범위

카카오 메시지 API의 `나에게 보내기`를 사용한다.

아래 기능은 1차 버전에 넣지 않는다.

- 카카오 채널 메시지
- 여러 사용자 발송
- 사용자 로그인

## 메시지 내용

카카오 메시지에는 아래 내용을 담는다.

- 오늘의 제목
- 면접 질문 3개
- 짧은 답변 예시
- 가능하면 주제와 난이도 라벨

## 실패 처리

- 발송 실패 시 1번 재시도한다.
- 최종 성공 또는 실패 상태를 MySQL에 저장한다.
- 가능한 경우 에러 메시지도 저장한다.

## 비밀키 규칙

카카오 REST API 키, 액세스 토큰, 리프레시 토큰은 절대 커밋하지 않는다.
