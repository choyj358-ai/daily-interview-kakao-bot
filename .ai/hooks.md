# Hook 아이디어

아래 내용은 아직 활성화된 코드가 아니라, 추후 자동화 후보이다.

## 커밋 전

나중에 자동으로 확인할 수 있는 항목:

- `.\gradlew.bat test` 실행
- 비밀키가 커밋에 포함되지 않았는지 확인
- `.env` 파일이 Git에 올라가지 않는지 확인

## 매일 질문 생성 후

나중에 자동으로 확인할 수 있는 항목:

- 생성된 질문 개수 기록
- 카카오 발송 상태 저장
- 발송 실패 시 1번 재시도

## 배포 전

나중에 자동으로 확인할 수 있는 항목:

- 필요한 환경변수가 모두 있는지 확인
- DB 연결 확인
- 카카오 토큰 유효성 확인
- OpenAI 모델명이 설정되어 있는지 확인
